מחולל פק"א — חיבור אוטומטי לגוגל קלנדר

בא"פ מרכז 8222 · ביה"ס ללחימה · צוות מרכזי — מושך אירועי שבוע אימון מהיומן ומייצא את הלו"ז (יומי / מקביל / פלס"ם) בפורמט התבנית.

1 · חיבור לגוגל קלנדר

מזהה מובנה — אין צורך לשנות. פשוט לחץ "התחבר לחשבון Google". (אפשר לעקוף עם מזהה אחר במידת הצורך.)
לא מחובר
בחיבור הראשון Google עשויה להציג מסך "Google hasn't verified this app" — זה תקין. לחץ Advanced (מתקדם) → Go to yahelmanor20.github.io (unsafe) → Continue, ואז אשר גישת קריאה בלבד ליומן. שום מידע לא נשלח לשרת חיצוני — הכול רץ בדפדפן שלך.

2 · בחירת היומן והשבוע

אם היומן הרצוי לא מופיע ברשימה — הדבק כאן את כתובת היומן. שדה זה גובר על הבחירה למעלה.
מס' הלוחמים/פלס"ם משמשים לחישוב התחמושת בדרישות (4 מחסניות/לוחם + 1/פלס"ם). אופציונלי — בלעדיהם תוצג הנוסחה למילוי.

3 · תצוגה מקדימה של הלו"ז

לאחר משיכת האירועים יוצג כאן הלו"ז המפורט לפי ימים.

4 · ייצוא לו"ז (Word)

הקובץ כולל את הלו"ז היומי, לו"ז מקביל, לו"ז פלס"ם (בפורמט התבנית), ובסוף — נספח דרישות אימון נגזר אוטומטית (לרוחב). שאר הסעיפים המילוליים ממולאים ידנית.

חיבור ראשון — מה לצפות

  1. לחץ "התחבר לחשבון Google" ובחר את חשבון ה-Google שלך.
  2. אם מופיע מסך "Google hasn't verified this app" — לחץ Advanced → Go to yahelmanor20.github.io (unsafe) → Continue. זה תקין.
  3. אשר את הרשאת הקריאה ליומן (calendar.readonly).
  4. בחר יומן ושבוע → "משוך אירועים" → "צור והורד פק"א".
החיבור דורש הרשאת קריאה בלבד ליומן (calendar.readonly). שום מידע אינו נשלח לשרת חיצוני — הכול רץ בדפדפן.
הגדרת מפתח (חד-פעמי, לא נדרש למפקצים)
  1. console.cloud.google.com → צור פרויקט.
  2. APIs & Services → Library → הפעל Google Calendar API.
  3. OAuth consent screen → External → Publish (Production).
  4. Credentials → OAuth client ID → Web application.
  5. Authorized JavaScript origins → https://yahelmanor20.github.io.
  6. העתק את ה-Client ID ל-DEFAULT_CLIENT_ID בקוד (או לשדה למעלה כ-override).
