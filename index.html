מחולל פק"א — חיבור אוטומטי לגוגל קלנדר

בא"פ מרכז 8222 · ביה"ס ללחימה · צוות מרכזי — מושך אירועי שבוע אימון מהיומן ומייצא את הלו"ז (יומי / מקביל / פלס"ם) בפורמט התבנית.

1 · חיבור לגוגל קלנדר

מזהה חד-פעמי שמפיקים בקונסולת Google Cloud. נשמר מקומית בדפדפן בלבד. הסבר מלא בתחתית העמוד.
לא מחובר

2 · בחירת היומן והשבוע

אם היומן הרצוי לא מופיע ברשימה — הדבק כאן את כתובת היומן. שדה זה גובר על הבחירה למעלה.

3 · תצוגה מקדימה של הלו"ז

לאחר משיכת האירועים יוצג כאן הלו"ז המפורט לפי ימים.

4 · ייצוא לו"ז (Word)

הקובץ כולל אך ורק את הלו"ז היומי, לו"ז מקביל ולו"ז פלס"ם — בפורמט זהה לתבנית. שאר סעיפי הפק"א ממולאים ידנית.

הגדרת חיבור Google (חד-פעמי)

  1. היכנס ל-console.cloud.google.com וצור פרויקט.
  2. ב-APIs & Services → Library, הפעל את Google Calendar API.
  3. ב-OAuth consent screen הגדר אפליקציה (External) והוסף את עצמך כ-Test user.
  4. ב-Credentials → Create credentials → OAuth client ID → סוג Web application.
  5. תחת Authorized JavaScript origins הוסף את הכתובת שבה רץ הקובץ (למשל http://localhost:8000 או דומיין הארגון).
  6. העתק את ה-Client ID והדבק בשדה למעלה.
החיבור דורש הרשאת קריאה בלבד ליומן (calendar.readonly). שום מידע אינו נשלח לשרת חיצוני — הכול רץ בדפדפן.
