הגדרת חיבור Google (חד-פעמי)

  1. היכנס ל-console.cloud.google.com וצור פרויקט.
  2. ב-APIs & Services → Library, הפעל את Google Calendar API.
  3. ב-OAuth consent screen הגדר אפליקציה (External) והוסף את עצמך כ-Test user.
  4. ב-Credentials → Create credentials → OAuth client ID → סוג Web application.
  5. תחת Authorized JavaScript origins הוסף את הכתובת שבה רץ הקובץ (למשל http://localhost:8000 או דומיין הארגון).
  6. העתק את ה-Client ID והדבק בשדה למעלה.
החיבור דורש הרשאת קריאה בלבד ליומן (calendar.readonly). שום מידע אינו נשלח לשרת חיצוני — הכול רץ בדפדפן.
