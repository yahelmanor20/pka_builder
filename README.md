# פרויקט פק"א / דרישות אימון

כלים להפקת **פק"א (פקודת אימון)** ו**דרישות אימון** שבועיות מנתוני Google Calendar,
כמסמכי Word בעברית (RTL). בא"פ 8222, ביה"ס ללחימה.

## מה יש כאן

| פריט | תיאור |
|------|--------|
| `index.html` | מחולל הפק"א. נפתח בדפדפן, מתחבר ל-Google Calendar (OAuth, קריאה בלבד), מושך אירועי שבוע ומייצא פק"א ל-Word. |
| `CLAUDE.md` | כל ההקשר, הכללים והמיפויים — נקרא אוטומטית ע"י Claude Code. |
| `docs/` | מפרט מבנה הפק"א + פורמט הדרישות. |
| `examples-pka/` | פק"אות אמת שכבר הופקו. |
| `examples-luz/` | לו"זים של אימונים שקרו (מקבילה ביומן מפ"ץ מרכזי). |
| `source-images/` | צילומי רפרנס. |

## הרצה עם Claude Code

```bash
cd pka-project
claude
```

Claude Code יקרא את `CLAUDE.md` אוטומטית. כדי לבדוק את המחולל בדפדפן — פתח את `index.html`.
